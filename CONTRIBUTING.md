# Contributing

Thanks for helping improve `clawfile`.

## Development

```bash
npm i
npm test
npm run lint
node bin/clawfile.js install Clawfile.example --dry-run
```

## Pull Requests

- Keep changes focused and small.
- Add/update tests when behavior changes.
- Update README/RELEASING docs when CLI flags or workflow changes.
